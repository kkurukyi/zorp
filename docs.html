Zorp Docs

Everything you need to set up, configure, and extend Zorp — structured memory infrastructure for autonomous AI agents.

Quick Start

Get up and running in under a minute. Clone, init, and start remembering.

# Clone the repository git clone https://github.com/zorp-code/zorp.git cd zorp # Initialize a new vault zorp init ~/memory --name my-brain # Start a session zorp wake # Store a decision zorp remember decision "Use PostgreSQL" --content "Chosen for JSONB" # Search zorp search "postgresql" zorp vsearch "what did we decide about storage" # End session zorp sleep "finished auth plan" --next "implement migration"

Vault Structure

Every Zorp vault follows a consistent directory layout. Memories are organized by type for easy browsing and Obsidian compatibility.

vault/ ├── .zorp/ # Internal state & config ├── decisions/ # Key architectural choices ├── lessons/ # Insights & learnings ├── people/ # Contact & relationship data ├── projects/ # Active project tracking ├── handoffs/ # Session transition notes ├── inbox/ # Quick captures └── templates/ # Document templates

Observation Format

Raw transcripts pass through the observation pipeline and become scored, typed entries. Each observation carries a confidence (c) and importance (i) score.

# Each observation is scored for confidence (c) and importance (i) - [decision|c=0.9|i=0.8] Use PostgreSQL for primary datastore - [lesson|c=0.7|i=0.6] Batch inserts outperform individual writes by 12x - [fact|c=1.0|i=0.4] API rate limit is 1000 req/min

Sessions

Sessions are the heartbeat of agent memory. Wake to load context, checkpoint to save progress, sleep to persist everything.

zorp wake # Start session zorp checkpoint --working-on "feature X" # Save progress zorp context "authentication" # Retrieve context zorp sleep "completed auth" --next "tests" # End session zorp recover # Recover interrupted zorp handoff --to agent-02 # Hand off

Integrations

Zorp plugs into your existing workflow. First-class support for Obsidian, OpenClaw, and remote sync.

Obsidian

Graph themes (neural, minimal), canvas dashboards, bidirectional kanban sync. Run zorp setup --theme neural.

OpenClaw

First-class hook pack. Install with openclaw hooks install zorp. Auto session lifecycle and context injection.

Tailscale + WebDAV

Serve vault over Tailscale for mobile sync. Run zorp tailscale-serve --vault ~/memory.

Replay Sources

Ingest past conversations from any major AI platform. Transcripts become searchable, structured memory.

zorp replay --source claude # Anthropic Claude zorp replay --source chatgpt # OpenAI ChatGPT zorp replay --source opencode # OpenCode sessions zorp replay --source openclaw # OpenClaw transcripts
